# SURFMORE SEO Audit

Omfattende tekniske SEO-audit: hele sitet via sitemap eller kun forsiden. Tjekker titel, meta, H1–H3, viewport, billeder (alt, mål), links, canonical, Open Graph, Twitter, JSON-LD, indholdslængde, URL, robots.txt, HTTPS m.m.

## Lokalt

```bash
npm install
npm run dev
```

Åbn http://localhost:3001

## Deploy på Vercel

1. Gå til [vercel.com](https://vercel.com) → **Add New** → **Project**.
2. Vælg **Import Git Repository** og forbind **SurfmoreSEO** (eller dit repo).
3. Vercel genkender automatisk **Next.js** fra `package.json` – ændr ikke Framework til noget andet.
4. Klik **Deploy**.

**Vigtigt:** Importer det eksisterende GitHub-repo. Opret ikke et nyt "Next.js"-projekt fra template – det er ikke samme ting.

Projektet er bygget med **Next.js 14** (App Router).
